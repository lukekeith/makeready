Create a Pinia composable store `useMemberDetailUI` in `resources/js/islands/admin-island/stores/ui/member-detail.ui.ts`.

State:
- `isOpen: ref(false)` — drawer visibility
- `selectedMemberId: ref<string | null>(null)` — currently selected member userId
- `profile: ref<MemberProfile | null>(null)` — loaded profile data
- `enrollmentProgress: ref<Array<{ programName: string; progressPercentage: number; completedLessons: number; totalLessons: number }>>([])`
- `isLoadingProfile: ref(false)`
- `isLoadingEnrollments: ref(false)`
- `error: ref<string | null>(null)`

Actions:
- `openDrawer(userId: string): void` — sets `selectedMemberId`, sets `isOpen = true`, calls `loadProfile()` and `loadEnrollmentProgress()` in parallel.
- `closeDrawer(): void` — sets `isOpen = false`, after 200ms delay (transition), resets `selectedMemberId`, `profile`, `enrollmentProgress` to null/empty.
- `async loadProfile(): Promise<void>` — calls `membersDomain.loadMemberProfile(selectedMemberId)`, sets `profile`.
- `async loadEnrollmentProgress(): Promise<void>` — fetches `GET /admin/api/members/${selectedMemberId}/enrollments` via axios. Response expected shape: `{ enrollments: [{ studyProgram: { name }, progressPercentage, completedLessons, totalLessons }] }`. Maps response into `enrollmentProgress` array. If endpoint 404s or fails, set `enrollmentProgress = []` gracefully (not all members have enrollments).

Computed:
- `displayName: computed(() => profile ? [profile.firstName, profile.lastName].filter(Boolean).join(' ') || 'Unknown' : '')`
- `avatarUrl: computed(() => profile?.profilePicture || profile?.googlePicture || null)`
- `initials: computed(() => { ... })` — first letter of firstName + first letter of lastName, uppercase. Fallback: '?'
- `joinedDate: computed(() => ...)` — earliest `joinedAt` from profile.groups, formatted as "Joined Mar 15, 2026"
- `groupsWithMeta: computed(() => profile?.groups ?? [])` — pass-through for now, Plan 02 adds add/remove logic

Import `useMembersDomain` from domain stores. Import axios for enrollment endpoint.

**Component: `resources/js/components/admin/member-profile-drawer/member-profile-drawer.vue`**

Create a Vue SFC that uses `Teleport` to render a slide-over drawer to `body`. Uses `useMemberDetailUI` store directly (this is an admin island component, not a reusable UI component).

Template structure (BEM block `MemberProfileDrawer`):
```
<Teleport to="body">
  <Transition name="drawer">
    <div v-if="store.isOpen" class="MemberProfileDrawer">
      <!-- Backdrop -->
      <div class="MemberProfileDrawer__backdrop" @click="store.closeDrawer()" />

      <!-- Panel -->
      <div class="MemberProfileDrawer__panel">
        <!-- Header -->
        <div class="MemberProfileDrawer__header">
          <div class="MemberProfileDrawer__avatar">
            <img v-if="store.avatarUrl" :src="store.avatarUrl" alt="" />
            <div v-else class="MemberProfileDrawer__avatar-placeholder">{{ store.initials }}</div>
          </div>
          <div class="MemberProfileDrawer__header-info">
            <h2 class="MemberProfileDrawer__name">{{ store.displayName }}</h2>
            <span class="MemberProfileDrawer__joined">{{ store.joinedDate }}</span>
          </div>
          <button class="MemberProfileDrawer__close" @click="store.closeDrawer()">
            <X :size="20" />
          </button>
        </div>

        <!-- Loading state -->
        <div v-if="store.isLoadingProfile" class="MemberProfileDrawer__loading">Loading...</div>

        <template v-else-if="store.profile">
          <!-- Contact info section -->
          <div class="MemberProfileDrawer__section">
            <h3 class="MemberProfileDrawer__section-title">Contact</h3>
            <div class="MemberProfileDrawer__contact-row" v-if="store.profile.phoneNumber">
              <span class="MemberProfileDrawer__contact-label">Phone</span>
              <span class="MemberProfileDrawer__contact-value" @click="copyToClipboard(store.profile.phoneNumber)" title="Click to copy">
                {{ store.profile.phoneNumber }}
                <Copy :size="14" class="MemberProfileDrawer__copy-icon" />
              </span>
            </div>
            <div class="MemberProfileDrawer__contact-row" v-if="store.profile.email">
              <span class="MemberProfileDrawer__contact-label">Email</span>
              <span class="MemberProfileDrawer__contact-value" @click="copyToClipboard(store.profile.email)" title="Click to copy">
                {{ store.profile.email }}
                <Copy :size="14" class="MemberProfileDrawer__copy-icon" />
              </span>
            </div>
          </div>

          <!-- Groups section -->
          <div class="MemberProfileDrawer__section">
            <h3 class="MemberProfileDrawer__section-title">Groups</h3>
            <div class="MemberProfileDrawer__groups">
              <div v-for="group in store.groupsWithMeta" :key="group.id" class="MemberProfileDrawer__group-chip">
                <span class="MemberProfileDrawer__group-name">{{ group.name }}</span>
                <span class="MemberProfileDrawer__group-role">{{ group.role }}</span>
                <!-- Plan 02 adds X button for remove -->
              </div>
            </div>
            <!-- Plan 02 adds "Add to group" button here -->
          </div>

          <!-- Enrollment progress section -->
          <div class="MemberProfileDrawer__section">
            <h3 class="MemberProfileDrawer__section-title">Enrollments</h3>
            <div v-if="store.isLoadingEnrollments" class="MemberProfileDrawer__loading">Loading enrollments...</div>
            <div v-else-if="store.enrollmentProgress.length === 0" class="MemberProfileDrawer__empty">No enrollments</div>
            <div v-else class="MemberProfileDrawer__enrollments">
              <div v-for="(enrollment, idx) in store.enrollmentProgress" :key="idx" class="MemberProfileDrawer__enrollment">
                <span class="MemberProfileDrawer__enrollment-name">{{ enrollment.programName }}</span>
                <div class="MemberProfileDrawer__progress-bar">
                  <div class="MemberProfileDrawer__progress-fill" :style="{ width: enrollment.progressPercentage + '%' }" />
                </div>
                <span class="MemberProfileDrawer__progress-text">
                  {{ enrollment.completedLessons }} of {{ enrollment.totalLessons }} lessons
                </span>
              </div>
            </div>
          </div>
        </template>
      </div>
    </div>
  </Transition>
</Teleport>
```
