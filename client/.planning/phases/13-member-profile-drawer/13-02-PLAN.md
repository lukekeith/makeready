**1. Update member-detail.ui.ts — add group management actions:**

Add these imports: `useGroupsDomain`, `useAllMembersDomain`, and axios.

New state:
- `isAddingToGroup: ref(false)` — loading state during add
- `isRemovingFromGroup: ref(false)` — loading state during remove

New computed:
- `availableGroups: computed(() => { ... })` — returns `groupsDomain.groups` filtered to exclude groups the member is already in (compare by group.id against profile.groups[].id). Return array of `{ id: string, name: string }`.

New actions:
- `async addToGroup(groupId: string): Promise<void>` — sets `isAddingToGroup = true`, calls `POST /admin/api/groups/${groupId}/members` with body `{ userId: selectedMemberId }` via axios. On success: calls `membersDomain.loadMembers(groupId, true)` to refresh that group's cache, then re-fetches the profile via `loadProfile()` to update the drawer's group list, then calls `allMembersDomain.loadAll()` to refresh the unified table. Sets `isAddingToGroup = false`. On error: set error, set isAddingToGroup false.
- `async removeFromGroup(groupId: string): Promise<void>` — sets `isRemovingFromGroup = true`. Must find the member's membership ID for that group: look up `membersDomain.membersByGroup[groupId]` to find the entry matching `selectedMemberId` by userId, get its `id`. Call `membersDomain.removeMember(groupId, membershipId)`. Then: `membersDomain.loadMembers(groupId, true)`, re-fetch profile via `loadProfile()`, call `allMembersDomain.loadAll()`. Sets `isRemovingFromGroup = false`.

Add local state for confirm dialog:
```typescript
const showRemoveConfirm = ref(false)
const removeTarget = ref<{ groupId: string; groupName: string } | null>(null)
const addGroupValue = ref<string | undefined>(undefined)

function confirmRemoveGroup(group: { id: string; name: string }) {
  removeTarget.value = { groupId: group.id, groupName: group.name }
  showRemoveConfirm.value = true
}

async function handleConfirmRemove() {
  if (!removeTarget.value) return
  await store.removeFromGroup(removeTarget.value.groupId)
  showRemoveConfirm.value = false
  removeTarget.value = null
}

function handleCancelRemove() {
  showRemoveConfirm.value = false
  removeTarget.value = null
}

async function handleAddGroup(groupId: string | undefined) {
  if (!groupId) return
  await store.addToGroup(groupId)
  addGroupValue.value = undefined
}
```

Mount AdminConfirmDialog inside the drawer panel:
```html
<AdminConfirmDialog
  :open="showRemoveConfirm"
  :title="`Remove from ${removeTarget?.groupName ?? 'group'}?`"
  :message="`Remove ${store.displayName} from ${removeTarget?.groupName}?`"
  confirm-label="Remove"
  :dangerous="true"
  @confirm="handleConfirmRemove"
  @cancel="handleCancelRemove"
/>
```
