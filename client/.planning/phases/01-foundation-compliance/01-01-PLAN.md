    6. Copy `.planning/` and `TWILIO_A2P_COMPLIANCE.md` back into the repo root

    **Step 2: Initialize Laravel 12**
    1. Run `composer create-project laravel/laravel temp-laravel "12.*" --prefer-dist` in a temp directory
    2. Move all files from temp-laravel into repo root (careful not to overwrite `.git/` or `.planning/`)
    3. Remove temp-laravel directory
    4. Set `DB_CONNECTION=sqlite` and `DB_DATABASE=:memory:` in `.env` and `.env.example`
    5. Remove all `DB_*` lines except DB_CONNECTION and DB_DATABASE from `.env.example`

    **Step 3: Configure Vite for Vue + SCSS**
    1. Install: `npm install --save-dev @vitejs/plugin-vue sass`
    2. Replace `vite.config.js` with the pattern from RESEARCH.md Pattern 2:
       - Input: `['resources/css/app.scss', 'resources/js/app.js']`
       - Vue plugin with `transformAssetUrls` config
       - Resolve alias: `vue: 'vue/dist/vue.esm-bundler.js'`
    3. Rename `resources/css/app.css` to `resources/css/app.scss`
    4. Update `resources/js/app.js` to import Vue's `createApp` (minimal — just mount point for future islands)

    **Step 4: Configure environment**
    1. Add to `config/services.php`: `'makeready' => ['url' => env('API_URL', 'https://api.makeready.org')]`
    2. Add to `.env.example` all vars from RESEARCH.md "Required Environment Variables" section:
       APP_KEY, APP_ENV, APP_DEBUG, APP_URL, API_URL, LOG_CHANNEL, SESSION_DRIVER, SESSION_SECURE_COOKIE, SESSION_SAME_SITE, DB_CONNECTION, DB_DATABASE
    3. Set SESSION_DRIVER=cookie in `.env` and `.env.example`
    4. Set LOG_CHANNEL=stderr in `.env.example` (production default)

    **Step 5: Create shared Blade layout and landing page**
    1. Create `resources/views/layouts/app.blade.php` per RESEARCH.md Pattern 7 — includes `@vite()` directive, navigation partial, footer partial, `@yield('content')`
    2. Create minimal `resources/views/partials/navigation.blade.php` (placeholder nav — will be styled in Phase 2)
    3. Create minimal `resources/views/partials/footer.blade.php` (placeholder footer)
    4. Create `resources/views/home.blade.php` extending layout with "Welcome to MakeReady" content
    5. Update `routes/web.php`: `Route::get('/', fn () => view('home'));`
    6. In `bootstrap/app.php`, configure `encryptCookies(except: [...])` per RESEARCH.md Pattern — use placeholder cookie names for now (will be confirmed in Plan 02)

    **Step 6: Remove Tailwind if present**
    Laravel 12 may include Tailwind by default. If `tailwind.config.js` or `postcss.config.js` with tailwind exists:
    1. Delete `tailwind.config.js`
    2. Remove tailwindcss from `postcss.config.js` or delete the file
    3. Run `npm uninstall tailwindcss postcss autoprefixer` if they are in package.json
    4. Ensure `resources/css/app.scss` contains no `@tailwind` directives
