**PreviewController** (app/Http/Controllers/PreviewController.php):
- Constructor injects ApiService
- `studyPreview(Request $request, string $token)`:
  - Fetch from `/public/preview/{$token}` via ApiService (no memberId needed -- public endpoint)
  - Extract program/study data and lessons list from response body
  - abort(404) if status !== 200
  - Render `pages.study-preview` with compact('token', 'studyData', 'lessons')
  - Forward Set-Cookie headers

- `lessonPreview(Request $request, string $token, string $lessonId, int $step = 1)`:
  - Fetch from `/public/preview/{$token}/lesson/{$lessonId}` via ApiService
  - Extract lesson data from response body
  - abort(404) if status !== 200
  - Render `pages.lesson-preview` with compact('token', 'lessonId', 'lessonData', 'step')
  - Forward Set-Cookie headers

**study-preview.blade.php**:
- Extends layouts.app (public page, no Navigation)
- Section title: study name from $studyData
- `<main class="StudyPreviewPage">`
- Display study/program name and description
- Loop through lessons rendering lesson cards with links to `/public/preview/{$token}/lesson/{lessonId}`
- BEM classes: StudyPreviewPage, StudyPreviewPage__header, StudyPreviewPage__lessons

**lesson-preview.blade.php**:
- Extends layouts.app (public page)
- Section title: lesson name from $lessonData
- `<main class="LessonPreviewPage">`
- Mount LessonIsland with isPreview=true:
  ```
  <div data-vue="LessonIsland" data-props="{{ json_encode([
      'lessonData' => $lessonData,
      'groupId' => '',
      'lessonScheduleId' => '',
      'initialStep' => (int) $step,
      'isPreview' => true,
  ]) }}"></div>
  ```
- groupId and lessonScheduleId are empty strings in preview mode (island should handle this gracefully -- AJAX save routes are disabled when isPreview=true)

**Routes** (web.php) -- outside auth group (public):
- `Route::get('/public/preview/{token}', [PreviewController::class, 'studyPreview'])->name('preview.study')`
- `Route::get('/public/preview/{token}/lesson/{lessonId}/{step?}', [PreviewController::class, 'lessonPreview'])->name('preview.lesson')`
- Import PreviewController at top of web.php
