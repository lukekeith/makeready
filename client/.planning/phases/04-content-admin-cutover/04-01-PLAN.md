**LessonController** (app/Http/Controllers/LessonController.php):
- Constructor injects ApiService
- `show(Request $request, string $groupId, string $lessonScheduleId, int $step = 1)`: Fetches `/api/member/lessons/{$lessonScheduleId}?memberId={$memberId}`, renders `pages.lesson` with lessonData, groupId, lessonScheduleId, step, isPreview=false. abort(404) on non-200.
- `submitNote(Request $request, string $groupId, string $lessonScheduleId, string $activityId)`: Proxies POST to `/api/member/activities/{$activityId}/submit` with request JSON body. Returns JSON response with forwarded cookies.
- `saveVideoProgress(Request $request, string $groupId, string $lessonScheduleId, string $activityId)`: Proxies POST to `/api/member/activities/{$activityId}/video-progress`. Returns JSON.
- `fetchScripture(Request $request, string $translation, string $book, string $chapter)`: Proxies GET to `/api/bible/{$translation}/{$book}/{$chapter}`. Returns JSON. This is a public route (no auth needed).
- All proxy methods forward cookies and return JSON responses (not Blade views).

**Blade shell** (resources/views/pages/lesson.blade.php):
- Extends layouts.home
- Section title: lesson name from $lessonData
- Contains a single `<main class="LessonPage">` with `<div data-vue="LessonIsland" data-props="...">` passing lessonData, groupId, lessonScheduleId, initialStep (cast to int), isPreview as JSON.

**LessonIsland** (resources/js/components/domain/lesson-island/lesson-island.vue):
- Props: `lessonData` (Object), `groupId` (String), `lessonScheduleId` (String), `initialStep` (Number, default 1), `isPreview` (Boolean, default false)
- Reactive state: `currentStepNumber = ref(props.initialStep)`, `lesson = ref(props.lessonData)`, `isLoading = ref(false)`, `isSaving = ref(false)`
- Computed `steps`: Build flat step list from `lesson.value.activities` array. Each activity maps to one or more steps based on activity type. VIDEO activities -> video-step, READ activities -> read-step, USER_INPUT activities -> input-step. After all activities, add a COMPLETE step.
- Computed `currentStep`: `steps.value[currentStepNumber.value - 1]`
- Computed `totalSteps`: `steps.value.length`
- Methods:
  - `goToStep(n)`: sets currentStepNumber, calls `history.pushState` to update URL to `/groups/{groupId}/lessons/{lessonScheduleId}/{n}`
  - `nextStep()`: goToStep(currentStepNumber + 1) if not at end
  - `prevStep()`: goToStep(currentStepNumber - 1) if not at start
  - `submitNote(activityId, noteType, content)`: POST to `/groups/${groupId}/lessons/${lessonScheduleId}/activity/${activityId}/submit` via axios with `{ lessonScheduleId, note: { type: noteType, content } }`. On success, replace `lesson.value` with `response.data.lesson` (critical: API returns updated lesson). Then nextStep().
  - `saveVideoProgress(activityId, progress)`: POST to `/groups/${groupId}/lessons/${lessonScheduleId}/activity/${activityId}/video-progress` via axios
- Template: Renders LessonPageHeader (as Blade component in shell, NOT in island), StepIndicator (can be inline in island), and a dynamic component switch on `currentStep.type` rendering video-step, read-step, input-step, or complete-step.
- Import and use BulletTextInput and VideoPlayer from existing component paths.

**video-step.vue**: Receives `activity` prop. Renders VideoPlayer with `src` from `activity.video.url` or `activity.videoUrl`. Emits `next` when user clicks continue. Shows video title/description.

**read-step.vue**: Receives `activity` prop + `groupId` + `lessonScheduleId`. Fetches scripture if activity has scripture references via axios GET to `/api/bible/{translation}/{book}/{chapter}`. Renders scripture text blocks. Emits `next` on continue click. Shows loading state while fetching.

**input-step.vue**: Receives `activity` prop + `isPreview` + `isSaving`. Renders BulletTextInput with v-model. Shows save button (disabled in preview mode). Emits `submit` with (activityId, noteType, content). The noteType comes from `activity.noteType` or derive from activity `helpTitle` (e.g., "Observation" -> "OBSERVATION"). Pre-populates with existing note content from `activity.note.content` if available.

**complete-step.vue**: Shows completion message. Button to go back to study home (`/groups/{groupId}/study/{studyEnrollmentId}`). In preview mode, shows "Preview Complete" with link back.

**SCSS** (lesson-island.scss): BEM classes for LessonIsland, LessonIsland__step-container, LessonIsland__nav. Import in lesson-island.vue.
