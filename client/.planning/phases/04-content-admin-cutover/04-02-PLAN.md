**AdminController** (app/Http/Controllers/AdminController.php):
- `show(Request $request)` method
- Extract member from `$request->attributes->get('member')`
- **UNCONDITIONALLY check member role and abort(403) if not leader** (per user decision: "Protected by member.auth middleware + leader role check"). This is NOT optional -- every request must pass the leader gate:
  ```php
  $member = $request->attributes->get('member');
  if (($member['role'] ?? '') !== 'leader') {
      abort(403);
  }
  ```
- Per research: the React admin page is intentionally simple -- it shows the member's name, phone number, and a logout button. No CRUD, no organization management. Replicate exactly this.
- Render `pages.admin` with compact('member')

**admin.blade.php**:
- Extends layouts.home
- Section title: "Admin -- MakeReady"
- Navigation with selected=profile (or appropriate value)
- PageTitle "Admin"

- Display member phone: `{{ $member['phone'] ?? '' }}`
- Logout form: `<form method="POST" action="{{ route('logout') }}">@csrf <x-primitive.button type="submit" variant="Secondary">Log Out</x-primitive.button></form>`
- BEM classes: AdminPage, AdminPage__info, AdminPage__name, AdminPage__phone, AdminPage__actions
