**all-members.domain.ts:**

1. Define and export `UnifiedMember` interface:
```typescript
export interface UnifiedMember {
  userId: string
  name: string
  avatarUrl?: string
  groups: Array<{ groupId: string; groupName: string; role: string; joinedAt: string }>
  lastActive: string  // most recent joinedAt across all groups
}
```

2. Create `useAllMembersDomain` store using Pinia Composition API (`defineStore('all-members-domain', () => { ... })`).

3. Inside the store:
   - Compose with `useGroupsDomain()` and `useMembersDomain()` (call at top of setup function)
   - `isLoading = ref(false)` — true from loadAll() start until all groups settle
   - `failedGroups = ref<Array<{ groupId: string; groupName: string }>>([])` — groups whose API calls rejected
   - `allMembers = computed<UnifiedMember[]>(() => { ... })` — the core deduplication logic:
     * Read `membersDomain.membersByGroup` (all keys)
     * Read `groupsDomain.groups` to map groupId -> groupName
     * Build a `Map<string, UnifiedMember>` keyed by `userId`
     * For each group's members, if userId already in map, push to that entry's `groups` array; otherwise create new entry
     * Derive `lastActive` as the most recent `joinedAt` string (ISO comparison) across all groups for that userId
     * Return `[...map.values()]`
   - IMPORTANT: The computed must read from `membersDomain.membersByGroup` reactively. Since `membersByGroup` is a `ref<Record<...>>`, each per-group load mutates the record and Vue tracks it. The computed will re-evaluate as each group loads, giving progressive rendering.

4. `loadAll()` async function:
   - Set `isLoading.value = true`, clear `failedGroups.value = []`
   - Ensure groups are loaded: `if (groupsDomain.groups.length === 0) await groupsDomain.loadGroups()`
   - Fan out: `const results = await Promise.allSettled(groupsDomain.groups.map(g => membersDomain.loadMembers(g.id)))`
   - For each rejected result, find the corresponding group by index and push `{ groupId, groupName }` to `failedGroups`
   - Set `isLoading.value = false`
   - Note: Do NOT await each loadMembers sequentially. The Promise.allSettled fires all in parallel. As each resolves, `membersDomain.membersByGroup` updates reactively, which triggers `allMembers` computed to re-evaluate — this is the progressive loading behavior.

5. Export: `return { isLoading, failedGroups, allMembers, loadAll }`
