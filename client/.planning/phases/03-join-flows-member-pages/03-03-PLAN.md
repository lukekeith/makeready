  <action>
    **1. Create `app/Http/Controllers/MemberLoginController.php`:**

    Constructor: inject `ApiService $api`.

    **showPhone(Request $request):**
    - Render login page at phone step.
    - Return `view('pages.login', ['step' => 'phone'])`.

    **submitPhone(Request $request): JsonResponse**
    - Extract phone number from request.
    - Store phone in session: `session()->put('login.phone', $phone)`.
    - Call `$this->api->post('/api/members/verify-phone', ['phoneNumber' => $phone], $request)`.
    - Note: Login does NOT require organizationId (existing member already associated).
    - If API error, return JSON error.
    - Return `response()->json(['redirectUrl' => route('login.verify')])`.
    - Forward Set-Cookie headers.

    **showVerify(Request $request):**
    - Read phone from session.
    - If no phone in session, redirect to login.
    - Return `view('pages.login', ['step' => 'verify', 'phone' => session('login.phone')])`.

    **submitVerify(Request $request): JsonResponse**
    - Read phone from session, code from request.
    - Call `$this->api->post('/api/members/confirm-verification', ['phoneNumber' => $phone, 'code' => $code], $request)`.
    - If 200, clear login session data: `session()->forget('login')`.
    - Return `response()->json(['redirectUrl' => route('home')])`.
    - Forward Set-Cookie headers (CRITICAL â€” this response sets the API session cookie that authenticates future requests).

    **logout(Request $request): RedirectResponse**
    - Call `$this->api->post('/api/members/logout', [], $request)`.
    - Clear all session data: `$request->session()->flush()`.
    - Build redirect response to `/`.
    - Forward Set-Cookie headers from API (these clear the API session cookie).
    - Return redirect.
