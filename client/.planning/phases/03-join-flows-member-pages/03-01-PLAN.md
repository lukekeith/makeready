    AJAX wrapper around PhoneEntry (Option A from RESEARCH.md — wrapper island, not modifying original).
    Props: `ajaxSubmitUrl` (string), `title` (string, default "Enter your phone"), `secondaryButtonLabel` (string, optional), `secondaryRedirectUrl` (string, optional), `showSmsConsent` (boolean, default false), `privacyUrl` (string, optional), `termsUrl` (string, optional).

    Template: Renders `<PhoneEntry>` with title, secondaryButtonLabel, secondaryRedirectUrl passed through. Inside PhoneEntry's default slot, conditionally render SMS consent checkbox when `showSmsConsent` is true:
    ```html
    <label class="SmsConsent">
      <input type="checkbox" v-model="smsConsent" class="SmsConsent__checkbox" />
      <span class="SmsConsent__text">
        I agree to receive SMS messages. Msg & data rates may apply. Reply STOP to cancel.
        <a :href="privacyUrl">Privacy Policy</a> |
        <a :href="termsUrl">Terms</a>
      </span>
    </label>
    ```

    On PhoneEntry's submit event: POST to `ajaxSubmitUrl` with JSON body `{ phoneNumber, smsConsent }`, headers include `X-CSRF-TOKEN` from meta tag and `Accept: application/json`. On success JSON response with `redirectUrl`, do `window.location.href = redirectUrl`. On error, display error message in PhoneEntry's error slot or as reactive error ref. Disable submit button while request is in flight (loading state).

    **4. Create `resources/js/components/domain/join-verify-island/join-verify-island.vue`:**
    AJAX wrapper around VerifyCode.
    Props: `ajaxVerifyUrl` (string), `phone` (string, for display), `resendUrl` (string, optional — for re-sending code), `size` (string, default "Large"), `theme` (string, default "Light").

    Template: Shows phone number being verified, renders `<VerifyCode>` with size/theme/autoFocus. On VerifyCode's complete event: POST to `ajaxVerifyUrl` with JSON body `{ code }`, same CSRF/Accept headers. On success `redirectUrl`, navigate. On error, show error and reset the code input. Add a "Resend code" link that POSTs to `resendUrl` if provided.
