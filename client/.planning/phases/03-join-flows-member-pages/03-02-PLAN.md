    **1. Create `app/Http/Controllers/JoinController.php`:**

    Constructor: inject `ApiService $api`.

    **showEnterCode():** Return view `pages.join-code`.

    **submitCode(Request $request):** Read `code` from input, redirect to `route('join.group', ['code' => $code])`. (Simple form POST redirect.)

    **showStep(Request $request, string $code, ?string $step = null):**
    - Default `$step` to `'info'` if null
    - Valid steps: `['info', 'profile', 'phone', 'verify', 'confirmed']`
    - For `info` step: Call `$this->api->get("/api/groups/code/{$code}", $request)`. If 200, extract group data. Store `groupId` and `organizationId` in session. Pass `$group`, `$code`, `$step` to view. Forward Set-Cookie headers.
    - For `profile` step: Verify session has `join.{$code}.groupId` (redirect to info if missing). Pass `$code`, `$step`, and any existing session profile data to view.
    - For `phone` step: Verify session has profile data. Pass `$code`, `$step`, AJAX endpoint URL to view.
    - For `verify` step: Verify session has phone. Pass `$code`, `$step`, phone number, AJAX verify URL to view.
    - For `confirmed` step: Re-fetch group data for display. Pass `$code`, `$step`, group data to view.
    - Return `response()->view(...)` and forward any API Set-Cookie headers.

    **submitInfo(Request $request, string $code):**
    - Session should already have groupId/organizationId from showStep info.
    - Redirect to profile step: `redirect()->route('join.group', ['code' => $code, 'step' => 'profile'])`.

    **submitProfile(Request $request, string $code):**
    - Store first_name, last_name, gender, birthday in session under `join.{$code}.*`.
    - Redirect to phone step.

    **submitPhone(Request $request, string $code): JsonResponse**
    - MUST return JSON (this is an AJAX endpoint).
    - Check `$request->boolean('smsConsent')` — if false, return `response()->json(['error' => 'SMS consent is required'], 422)`.
    - Store phone in session.
    - Call `$this->api->post('/api/members/verify-phone', ['phoneNumber' => $phone, 'organizationId' => session("join.{$code}.organizationId")], $request)`.
    - If API returns non-200, return error JSON.
    - Return `response()->json(['redirectUrl' => route('join.group', ['code' => $code, 'step' => 'verify'])])`.
    - Forward Set-Cookie headers from API response.

    **submitVerify(Request $request, string $code): JsonResponse**
    - MUST return JSON.
    - Gather all session data: phone, firstName, lastName, gender, birthday, organizationId, groupId.
    - Call `$this->api->post('/api/members/confirm-verification', [...all fields + code from request...], $request)`.
    - If 200, also call `$this->api->post("/api/groups/{$groupId}/join-requests", [], $request)` to submit the join request.
    - Return `response()->json(['redirectUrl' => route('join.group', ['code' => $code, 'step' => 'confirmed'])])`.
    - Forward Set-Cookie headers.
    - On error, return error JSON with 422 status.

    IMPORTANT: For AJAX responses that need to forward Set-Cookie, attach cookies to the JsonResponse:
    ```php
    $response = response()->json(['redirectUrl' => '...']);
    foreach ($result['setCookies'] as $cookie) {
        $response->headers->set('Set-Cookie', $cookie, false);
    }
    return $response;
    ```
