- Initialize all three stores: `const domain = useProgramsDomain()`, `const listUI = useProgramsListUI()`, `const detailUI = useProgramDetailUI()`
- `const route = useRoute()`, `const router = useRouter()`
- `const isSaving = ref(false)`

onMounted:
```typescript
if (route.params.id) {
  await domain.getProgram(route.params.id as string)
} else {
  await Promise.all([domain.loadPrograms(), domain.loadTemplates()])
}
```
NOTE: Templates must be loaded eagerly so the create form's template select is populated.

watch(route.params.id):
```typescript
watch(
  () => route.params.id,
  async (newId) => {
    if (newId) {
      await domain.getProgram(newId as string)
    } else {
      await Promise.all([domain.loadPrograms(), domain.loadTemplates()])
    }
  }
)
```

Create form fields — MUST be computed (not const) because templateOptions changes after loadTemplates resolves:
```typescript
const createFields = computed(() => [
  { key: 'name', label: 'Program Name', type: 'text' as const, required: true, placeholder: 'Enter program name' },
  { key: 'description', label: 'Description', type: 'textarea' as const, placeholder: 'Describe this program (optional)' },
  { key: 'templateId', label: 'Template', type: 'select' as const, required: true, options: listUI.templateOptions },
  { key: 'days', label: 'Number of Days', type: 'number' as const, required: true, placeholder: '30' },
])
```

Edit fields (simple — no template or days):
```typescript
const editFields = [
  { key: 'name', label: 'Program Name', type: 'text' as const, required: true, placeholder: 'Enter program name' },
  { key: 'description', label: 'Description', type: 'textarea' as const, placeholder: 'Describe this program (optional)' },
]
```

Handler functions — mirror groups-section.vue:
- `handleCreate(payload)`: set isSaving, call `domain.createProgram(payload)`, call `listUI.closeForm()`, catch sets listUI.formError, finally resets isSaving
- `handleUpdate(payload)`: get editingProgram.id, set isSaving, call `domain.updateProgram(id, payload)`, call `listUI.closeForm()`, catch sets listUI.formError, finally resets isSaving
- `handleDelete()`: get confirmDeleteProgram.id, call `domain.deleteProgram(id)`, always call `listUI.cancelDelete()` in finally

Template (list view — when `!route.params.id`):
```html
<template v-else>
  <div class="AdminSection">
    <header class="AdminSection__header">
      <h1 class="AdminSection__title">Programs</h1>
      <button class="AdminSection__action-btn" @click="listUI.openCreateForm">
        + Create Program
      </button>
    </header>
    <div class="AdminSection__body">
      <AdminTable
        :columns="listUI.tableColumns"
        :rows="listUI.tableRows"
        :loading="domain.isLoading"
        empty-message="No programs yet. Create your first study program."
        @row-click="listUI.navigateToDetail"
        @edit="listUI.openEditForm"
        @delete="listUI.requestDelete"
      />
    </div>

    <!-- Create/Edit Form Modal -->
    <AdminForm
      :key="listUI.editingProgramId ?? 'create'"
      :open="listUI.isCreateFormOpen"
      :title="listUI.isEditing ? 'Edit Program' : 'Create Program'"
      :fields="listUI.isEditing ? editFields : createFields"
      :values="listUI.isEditing ? { name: listUI.editingProgram?.name ?? '', description: listUI.editingProgram?.description ?? '' } : {}"
      :error="listUI.formError ?? undefined"
      :saving="isSaving"
      @save="listUI.isEditing ? handleUpdate($event) : handleCreate($event)"
      @cancel="listUI.closeForm"
    />

    <!-- Delete Confirmation -->
    <AdminConfirmDialog
      :open="!!listUI.confirmDeleteProgram"
      title="Delete Program"
      :message="`Are you sure you want to delete &quot;${listUI.confirmDeleteProgram?.name ?? ''}&quot;? This will remove all lessons and enrollments. This cannot be undone.`"
      @confirm="handleDelete"
      @cancel="listUI.cancelDelete"
    />
  </div>
</template>
```
