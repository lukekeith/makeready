**programs.domain.ts** — mirrors groups.domain.ts structure exactly. Use the complete implementation from 07-RESEARCH.md Pattern 1:

- Export interfaces: `Program` (id, name, description?, coverImageUrl?, isPublished, days?, lessons?: Lesson[], _count?: { enrollments: number }), `Lesson` (id, dayNumber, title, programId), `Template` (id, name), `CreateProgramPayload` (name, templateId, days, description?, isPublished?), `UpdateProgramPayload` (name?, description?, isPublished?, days?, coverImageUrl?)
- Store ID: `'programs-domain'`
- Refs: `programs: ref<Program[]>([])`, `templates: ref<Template[]>([])`, `isLoading: ref(false)`, `error: ref<string | null>(null)`
- Methods (all wrap in try/catch, set error.value on failure):
  - `loadPrograms()`: GET `/admin/api/programs`, set `programs.value = res.data.programs ?? []`
  - `loadTemplates()`: GET `/admin/api/templates`. Cache: if templates.value.length > 0, return early (load once per session). Set `templates.value = res.data.templates ?? []`
  - `getProgram(id)`: GET `/admin/api/programs/${id}?lessonPage=1&lessonLimit=30`, upsert into programs array (findIndex + splice or push)
  - `createProgram(payload: CreateProgramPayload)`: POST `/admin/api/programs`, push `res.data.program` to programs.value. Re-throw on error.
  - `updateProgram(id, payload: UpdateProgramPayload)`: PATCH `/admin/api/programs/${id}`, splice-replace in programs.value. Re-throw on error.
  - `deleteProgram(id)`: DELETE `/admin/api/programs/${id}`, filter out. Re-throw on error.
  - `uploadCoverImage(id, file: File)`: POST with FormData (append 'image' key, NO explicit Content-Type header), update coverImageUrl in programs array. Re-throw on error.
  - `addLesson(programId)`: POST `/admin/api/programs/${programId}/lessons` with empty `{}`. Push returned lesson to program.lessons array.
  - `updateLessonTitle(programId, lessonId, title)`: PATCH `/admin/api/programs/${programId}/lessons/${lessonId}` with `{ title }`. Response is `{ success }` only — update lesson title directly in local state (find program, find lesson, set lesson.title = title). Do NOT parse a lesson from the response.
  - `deleteLesson(programId, lessonId)`: DELETE `/admin/api/programs/${programId}/lessons/${lessonId}`. After successful delete, call `await getProgram(programId)` to re-fetch with updated dayNumbers (server reassigns day numbers after deletion).
  - `reorderLessons(programId, lessonIds: string[])`: POST `/admin/api/programs/${programId}/reorder-lessons` with `{ lessonOrder: lessonIds }`. Response returns updated program — splice-replace in programs.value.
- Return all refs and methods
