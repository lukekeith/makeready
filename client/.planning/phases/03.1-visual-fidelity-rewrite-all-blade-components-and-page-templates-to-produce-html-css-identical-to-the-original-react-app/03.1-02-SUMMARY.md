### Constraint Deviations

**1. public-home container: `<main>` kept instead of `<div>`**
- **Found during:** Task 1
- **Issue:** React uses `<div class="PublicHomePage__container">` but `SsrHtmlTest` (existing test) asserts `<main>` tag must appear on GET /. Decision recorded in STATE.md from Phase 03-03.
- **Decision:** Keep `<main>` for container per prior architectural decision. The semantic element difference does not affect SCSS selector application.

**2. profile page: Navigation present in extends layout**
- The profile page extends `layouts.home` which may inject shared structure; Navigation removed from the page section content itself since React ProfilePage does not render Navigation.
