**members.domain.ts** — Pinia composition store (ID: `'members-domain'`):
- Interfaces: `GroupMember { id, userId, groupId, role: 'OWNER'|'ADMIN'|'MEMBER', name, avatarUrl?, joinedAt }`, `JoinRequest { id, status, message?, createdAt, member: { id, firstName?, lastName?, avatarUrl? } }`, `MemberProfile { id, firstName?, lastName?, phoneNumber, email?, profilePicture?, googlePicture?, groups: Array<{ id, name, coverImageUrl?, role, joinedAt }> }`
- State: `membersByGroup: ref<Record<string, GroupMember[]>>({})`, `requestsByGroup: ref<Record<string, JoinRequest[]>>({})`, `loadedGroupIds: ref(new Set<string>())`, `isLoading: ref(false)`, `error: ref<string|null>(null)`
- Methods:
  - `loadMembers(groupId, force = false)`: skip if loaded and !force; GET `/admin/api/groups/${groupId}/members`; set membersByGroup[groupId]; add to loadedGroupIds
  - `loadJoinRequests(groupId)`: GET `/admin/api/groups/${groupId}/join-requests`; set requestsByGroup[groupId]
  - `loadMemberProfile(memberId): Promise<MemberProfile>`: GET `/admin/api/members/${memberId}/profile`; return res.data.data
  - `approveRequest(groupId, requestId)`: POST `/admin/api/groups/${groupId}/join-requests/${requestId}/approve` body `{}`; filter from requestsByGroup; call `loadMembers(groupId, true)`
  - `rejectRequest(groupId, requestId)`: DELETE `/admin/api/groups/${groupId}/join-requests/${requestId}` (unconfirmed — wrap in try/catch, re-throw with message "Reject is not available"); filter from requestsByGroup
  - `changeRole(groupId, memberId, role)`: PATCH `/admin/api/groups/${groupId}/members/${memberId}` body `{ role }` (unconfirmed — wrap in try/catch); update local state
  - `removeMember(groupId, memberId)`: DELETE `/admin/api/groups/${groupId}/members/${memberId}` (unconfirmed — wrap in try/catch); filter from membersByGroup

**members-tab.ui.ts** — Pinia composition store (ID: `'members-tab-ui'`):

- Computeds:

- Methods:

  - `openProfile(memberId)`: set selectedMemberId, call domain.loadMemberProfile, set profileData, set isProfileOpen
  - `closeProfile()`: reset profileData, isProfileOpen, selectedMemberId
  - `requestRemove(memberId)`: set confirmRemoveId
  - `cancelRemove()`: reset confirmRemoveId
  - `confirmRemove()`: call domain.removeMember(groupId, confirmRemoveId), reset confirmRemoveId
  - `openRoleChange(memberId, currentRole)`: set roleChangeTarget
  - `confirmRoleChange(newRole)`: call domain.changeRole(groupId, roleChangeTarget.memberId, newRole), reset roleChangeTarget
