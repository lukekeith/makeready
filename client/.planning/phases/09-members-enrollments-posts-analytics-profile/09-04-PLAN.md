**posts.domain.ts** — Pinia composition store (ID: `'posts-domain'`):
- Types (export all):
  - `PostType = 'ANNOUNCEMENT' | 'POLL' | 'EVENT' | 'VIDEO' | 'WELCOME'`
  - `GroupPost { id, groupId, authorId?, authorName, authorAvatarUrl?, type: PostType, content, title?, pollOptions?: Array<{ id, text, voteCount, hasVoted }>, videoUrl?, eventDate?, eventLocation?, eventTitle?, viewCount?, shareCount?, createdAt, updatedAt }`
  - `CreatePostPayload { type: PostType, content: string, title?: string, pollOptions?: string[], videoUrl?: string, eventDate?: string, eventLocation?: string }`
- State: `postsByGroup: ref<Record<string, GroupPost[]>>({})`, `cursorByGroup: ref<Record<string, string|null>>({})`, `isLoading: ref(false)`, `error: ref<string|null>(null)`
- Methods:
  - `loadPosts(groupId, cursor?: string)`: GET `/admin/api/groups/${groupId}/posts?limit=20` + `&cursor=${cursor}` if cursor provided. If no cursor (first load), replace postsByGroup[groupId]; if cursor, append to existing. Set cursorByGroup[groupId] from res.data.nextCursor.
  - `createPost(groupId, payload: CreatePostPayload)`: POST `/admin/api/groups/${groupId}/posts` with payload. Prepend new post to postsByGroup[groupId].
- Return all.

**posts-tab.ui.ts** — Pinia composition store (ID: `'posts-tab-ui'`):
- Import `usePostsDomain`, `useRoute`
- Refs: `isCreateFormOpen: ref(false)`, `selectedType: ref<PostType>('ANNOUNCEMENT')`, `formError: ref<string|null>(null)`, `isSaving: ref(false)`
- Form refs (separate for each field — simpler than a single object for type-conditional rendering):
  - `formTitle: ref('')`, `formContent: ref('')`, `formVideoUrl: ref('')`, `formEventDate: ref('')`, `formEventLocation: ref('')`, `pollOptions: ref<string[]>(['', ''])`
- Computeds:
  - `groupId`: `route.params.id as string`
  - `posts`: `domain.postsByGroup[groupId] ?? []`
  - `hasMore`: `!!domain.cursorByGroup[groupId]`
  - `isLoading`: `domain.isLoading`
- Methods:
  - `loadData()`: call `domain.loadPosts(groupId)`
  - `loadMore()`: call `domain.loadPosts(groupId, domain.cursorByGroup[groupId])`
  - `openCreate()`: reset all form refs, set isCreateFormOpen true
  - `closeCreate()`: set isCreateFormOpen false
  - `addPollOption()`: push `''` to pollOptions
  - `removePollOption(index)`: splice pollOptions at index (min 2)
  - `buildPayload(): CreatePostPayload`: based on selectedType, build payload:
    - ANNOUNCEMENT: { type, content: formContent, title: formTitle || undefined }
    - POLL: { type, content: formContent, pollOptions: pollOptions.filter(o => o.trim()) }
    - EVENT: { type, content: formContent, title: formTitle || undefined, eventDate: formEventDate ? new Date(formEventDate).toISOString() : undefined, eventLocation: formEventLocation || undefined }
    - VIDEO: { type, content: formContent, title: formTitle || undefined, videoUrl: formVideoUrl }
  - `async submitCreate()`: set isSaving true, formError null; build payload via buildPayload(); call `domain.createPost(groupId, payload)`; on success call closeCreate(); on error set formError from err.response.data.message or fallback; finally set isSaving false.
- Return all.
