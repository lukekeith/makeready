# Link Railway CLI to MakeReady API

Run this command to link the Railway CLI to the MakeReady API service:

```bash
railway link -p b69d7c6d-dedf-44f9-a416-3949f32d2870 -s f5dc2315-9c49-4c85-b32d-99418eeba49d -e 47a94190-6ea6-48fd-b9db-9973f4f5912a
```

Then verify with:
```bash
railway status
```
