# no way to change enrollment calendar

## Problem
Once a group is enrolled in a study, the leader cannot change the enrollment's cadence — days of the week, time, or frequency. The server accepts no such edits and neither client has a screen for it. "Fixed" looks like: an enrollment-schedule management surface (edit cadence, reschedule) backed by a server capability to rewrite the enrollment's schedule. This is the UX-10 cluster — the largest UX item — and overlaps the "living studies" feature requests (12270300418/12354285433/12270300519).

## Reports (verbatim)
- (2026-06-13) "I want to change the days and time and frequency of the study but can't."
